GitHub Token Setup Guide

To add or edit robots in the encyclopedia, you need to set up a GitHub personal access token. This token allows the application to save data to your GitHub repository. Follow these steps to create and set up your token:

Quick Token Input

If you already have a GitHub token with the 'repo' scope, you can enter it here:

1 Go to your GitHub account

Sign in to your GitHub account at github.com. If you don't have an account yet, you'll need to create one.

2 Navigate to Personal Access Tokens

1. Click on your profile picture in the top-right corner
2. Select "Settings" from the dropdown menu
3. Scroll down and click on "Developer settings" in the left sidebar
4. Click on "Personal access tokens" in the left sidebar
5. Click on "Tokens (classic)"
6. Click the "Generate new token" button
7. If prompted, confirm your password

3 Create a new token

1. Give your token a descriptive name (e.g., "Tgen Robotics Hub")
2. Set an expiration date (e.g., 90 days or custom date)
3. In the "Select scopes" section, check the box next to "repo" to grant full access to repositories
4. Scroll down and click the "Generate token" button

Important: The "repo" scope is required for the application to save data to your GitHub repository. Make sure this option is selected.
4 Copy your token

After generating your token, GitHub will display it on the screen. This is the only time you will see the token, so make sure to copy it right away!

Security Note: Your token is like a password. Keep it secure and don't share it with others. If you lose your token, you'll need to generate a new one.
5 Add the token to the Tgen Robotics Hub

There are two ways to add your token to the application:

Option 1: Use the form at the top of this page to enter and save your token.

Option 2: Click on the "GitHub Token" button in the header of any page to open the token management dialog and enter your token.

Your token will be stored securely in your browser's localStorage, which means it stays on your computer and is not sent to any servers (except GitHub when making API calls).

6 Test your token

Once you've added your token, you can test it by:

1. Going to the Add Robot page
2. Filling out the form with a test robot
3. Clicking "Save Robot" to see if it saves successfully

If everything is working correctly, you should be able to see your new robot in the Encyclopedia.

Troubleshooting

If you're having issues saving robots, check the following:

  • Make sure your token has the 'repo' scope
  • Ensure your token has not expired
  • Check the browser console for any error messages (right-click > Inspect > Console)
  • Try clearing your token and adding it again
